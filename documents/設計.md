# REST API 一覧

## ユーザ登録API

### 概要
独自コインを利用するためのユーザ登録をおこなう。

登録時にユーザーを一意に表すユーザーIDが生成される。

### API基本情報

| 項目       | 内容        |
|----------|-----------|
| HTTPメソッド | POST      |
| リクエストURL | /api/user |

### リクエスト仕様
なし

### レスポンス仕様

#### 形式
| 文字コード        | UTF-8            |
|--------------|------------------|
| Content-Type | application/json |

#### 仕様
| 要素     | 必須  | 型   | 説明             |
|--------|-----|-----|----------------|
| userId | ○   | int | ユーザを一意に表すユーザID |

## 履歴API
出入金の履歴を取得する。

### API基本情報
| 項目       | 内容                        |
|----------|---------------------------|
| HTTPメソッド | POST                      |
| リクエストURL | /api/user/:userId/history |

### リクエスト仕様
なし

### レスポンス仕様

#### 形式
| 文字コード        | UTF-8            |
|--------------|------------------|
| Content-Type | application/json |

#### 仕様
| 要素                    | 必須  | 型      | 説明             |
|-----------------------|-----|--------|----------------|
| history               | ○   | Array  | 利用履歴のリスト       |
| history[N]/amount     | ○   | Long   | 出入金額           |
| history[N]/type       | ○   | String | 出入金種別（in, out） |
| history[N]/balance    | ○   | String | ユーザの残高         |
| history[N]/updateDate | ○   | Date   | 変更日時           |
